  <action>
Run `npm install @react-native-community/slider` in the android directory. Then run `npx pod-install ios` only if this is also targeting iOS (it is Android-only, so skip pod-install). The package resolves via autolinking for Android.

Create `android/src/hooks/useDetectionState.ts`. This hook owns all detection state shared between the badge and expanded card:
- `MAX_HISTORY = 10` constant
- `chunkHistory` state: array of `{ label: string; score: number }`, max 10 entries
- `threshold` state: `number`, default `0.7`
- `addResult(result: { label: string; score: number })` function: updates history using ring buffer pattern `[...prev.slice(-(MAX_HISTORY - 1)), result]`
- `getLabel(score: number, threshold: number): string` — if `score > threshold` return `'AI'`; if `score < (1 - threshold)` return `'HUMAN'`; else return `'UNCERTAIN'`
- `getExplanation(score: number): string` — score band templates: >0.85 = "Strong spectral artifacts consistent with TTS synthesis detected.", >0.65 = "Possible spectral artifacts consistent with AI-generated speech.", >0.45 = "Ambiguous signal — natural and synthetic characteristics mixed.", else "Natural voice characteristics detected. No synthesis artifacts found."
- Export all as a single `useDetectionState` hook

Create `android/src/components/HistoryDots.tsx`. Props: `history: { label: string; score: number }[]`. Renders a `View` with `flexDirection: 'row'` and `gap: 4`. Maps history to colored dots: AI = `#f59e0b`, HUMAN = `#22c55e`, UNCERTAIN = `#6b7280`. Fills remaining empty slots up to MAX_HISTORY with `#d1d5db` dots. Each dot is 10x10 with `borderRadius: 5`.

Create `android/src/components/ExpandedCard.tsx`. Receives props: `visible: boolean`, `score: number`, `label: string`, `history: { label: string; score: number }[]`, `threshold: number`, `onThresholdChange: (v: number) => void`, `onClose: () => void`.

Animate using `Animated.Value` initialized to 0. Use `Animated.spring` to animate to height 280 when `visible` becomes true, `Animated.timing` to animate to 0 when `visible` becomes false (duration 200ms), then call a callback to fully unmount if desired (optional — hiding at height 0 is acceptable).

The card content:
- `<Text>{Math.round(score * 100)}% AI-generated</Text>` (use `getExplanation` imported from the hook file, not re-implemented)
- `<Text>{explanation}</Text>`
- `<HistoryDots history={history} />`
- `<Slider value={threshold} onValueChange={onThresholdChange} minimumValue={0.5} maximumValue={0.95} step={0.05} />` from `@react-native-community/slider`
- `<TouchableOpacity onPress={onClose}><Text>✕</Text></TouchableOpacity>`

The `Animated.View` wraps all card content with `overflow: 'hidden'` so height animation clips the content cleanly.
